# AWI (Agrotronic Web Interface)
##### Web интерфейс системы Агротроник 
----
###### Стек:
 - ReactJS
 - NodeJS

https://agrotronic.rostselmash.com/newui - тест
https://agrotronic.rostselmash.com/ - продакшен

###### Старт
 - Установка пакетов: npm install или yarn install
 - Запуск npm start
 - Сборка стилей npm run build, для отслеживание изменений в стилях npm run watch-css
 - Линтёр стилей npm run stylelint, для автофиква npm run stylefix
 - Линтёр js npm run lint, для автофикса npm run lintfix
